# TP2 de Métodos Numéricos
## 2022, 2do cuatrimestre

## Instrucciones

## Compilación

1. Ir al src 

```
cd tp2-metnum-c2-2022/src
```

2. Compilar el programa

```
sh compile
``` 

## Correr el programa

```
./tp (nombre_archivo_test) (niter) (eps)
```

Ejemplo:

```
./tp ../tests/karateclub_matriz.txt 10000 0.00001
```
