# Reconocimiento de dígitos: TP2 de Métodos Numéricos
## 2022, 2do cuatrimestre

## Instrucciones

## Compilación

1. Ir al src 

```
cd tp2-metnum-c2-2022/src
```

2. Compilar el programa

```
python metnum.py
``` 

## Correr el programa

```
./tp (nombre_archivo_test) (niter) (eps)
```

Ejemplo:

```
./tp karateclub_matriz.txt 1000 0.0001
```
